Thanks for installing Flight JS snippets! Below are some quick notes to get you
started.

Supported snippets
==================

* Component and Mixin API
  - defineComponent (tab trigger: `flight`)
  - Component.attachTo (tab trigger: `attachTo`)
* Advice API (tab trigger: `this.`)
  - this.before
  - this.after
  - this.around
* Base API (tab trigger: `this.`)
  - this.defaultAttrs
  - this.select
  - this.on
  - this.off
  - this.trigger
